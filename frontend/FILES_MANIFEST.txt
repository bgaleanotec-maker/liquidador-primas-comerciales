ROUTES & FEATURES
==================

Route Mapping:
--------------
/                    -> Redirect to /dashboard
/login               -> Login page (public)
/dashboard           -> Dashboard (all authenticated users)
/variables           -> Variable governance (admin only)
/data-entry          -> Data ingestion (analyst+)
/liquidations        -> Liquidation management (all roles)
/approvals           -> Approval workflow (approver+)
/reports             -> Reporting (all roles)
/admin               -> Administration panel (admin only)

User Roles:
-----------
admin:    Full access
approver: Dashboard, liquidations, approvals, reports
analyst:  Dashboard, data entry, reports, liquidations

FORMATTING STANDARDS
====================

Currency:  Colombian Pesos (COP) with $ prefix
Date:      DD de MMMM de YYYY (Spanish locale)
Time:      HH:MM 24-hour format
Numbers:   . decimal separator, , thousands
Percent:   X.X% format

Example:
  $15.000.000 - 15 de Junio de 2024 - 10:30 - 92.5%

API INTEGRATION
===============

Base URL: /api (proxied to http://localhost:5000)

Auth Endpoints:
  POST   /api/auth/login          - Login
  POST   /api/auth/logout         - Logout
  GET    /api/auth/me             - Current user

Admin Endpoints:
  GET    /api/admin/users         - List users
  POST   /api/admin/users         - Create user
  PUT    /api/admin/users/{id}    - Update user
  DELETE /api/admin/users/{id}    - Delete user
  GET    /api/admin/audit         - Audit log
  GET    /api/admin/business-units
  POST   /api/admin/periods       - Create period
  GET    /api/admin/periods       - List periods
  GET    /api/admin/data-sources
  POST   /api/admin/data-sources

Variables Endpoints:
  GET    /api/variables/llaves    - List LLAVEs
  POST   /api/variables/llaves    - Create LLAVE
  PUT    /api/variables/llaves/{id}
  DELETE /api/variables/llaves/{id}
  GET    /api/variables/kpis
  POST   /api/variables/kpis
  PUT    /api/variables/kpis/{id}
  GET    /api/variables/structure/{bu_code}
  POST   /api/variables/validate-weights/{bu_code}

Data Endpoints:
  POST   /api/data/upload         - Upload CSV
  GET    /api/data/results        - Get KPI results
  PUT    /api/data/results/{id}   - Update result
  GET    /api/data/targets        - Get targets
  POST   /api/data/targets/upload
  GET    /api/data/template/{bu_code}

Liquidation Endpoints:
  GET    /api/liquidations        - List liquidations
  POST   /api/liquidations/calculate
  GET    /api/liquidations/{id}
  POST   /api/liquidations/{id}/submit

Approval Endpoints:
  GET    /api/approvals/pending
  POST   /api/approvals/{id}/approve
  POST   /api/approvals/{id}/reject
  GET    /api/approvals/history

Report Endpoints:
  GET    /api/reports/monthly-summary
  GET    /api/reports/bu-report
  GET    /api/reports/user-performance
  GET    /api/reports/trend
  POST   /api/reports/export

Metrics Endpoints:
  GET    /api/metrics/dashboard
  GET    /api/metrics/kpi-compliance
  GET    /api/metrics/period-progress
  GET    /api/metrics/source-coverage
